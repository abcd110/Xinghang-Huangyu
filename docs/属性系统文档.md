### 属性提升
- **公式**：最终值 = 强化后值 × (1.2 ^ 升华等级)
- **影响属性**：攻击、防御、生命
- **不受影响**：攻速、会心、暴伤、命中、闪避

## 七、属性计算公式

### 最终属性计算
```
基础属性 × (1 + 强化加成) × 升华倍率 + 其他加成
```

### 示例（攻击力）
```
基础攻击: 100
强化+10: 100 × (1 + 10×10%) = 200
升华+3:  200 × 1.2³ = 346
```
