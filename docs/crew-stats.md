# 船员数值设计文档

## 概述

本文档记录所有B/A/S级角色的基础数值和升级数值规则。

### 属性说明
- **HP**：生命值
- **Attack**：攻击力
- **Defense**：防御力
- **Speed**：攻击速度（每秒攻击次数）
- **CritRate**：暴击率（%）
- **CritDamage**：暴击伤害（%）

### 基础属性规则
所有船员的基础属性规则：
- **Speed**：固定为 1.0（不随升级变化）
- **CritRate**：固定为 5%（不随升级变化）
- **CritDamage**：固定为 150%（不随升级变化）

### 升级规则
每升1级，属性按以下比例提升：
- **HP**：+5%
- **Attack**：+3%
- **Defense**：+3%
- **Speed**：不变（固定1.0）
- **CritRate**：不变（固定5%）
- **CritDamage**：不变（固定150%）

### 稀有度差异
| 稀有度 | 基础属性倍率 | 最大等级 | 升级消耗系数 |
|--------|-------------|---------|-------------|
| B级 | 1.0x | 30 | 1.0 |
| A级 | 1.5x | 40 | 1.5 |
| S级 | 2.0x | 50 | 2.0 |

## 数值计算公式

### 等级属性计算
```
最终属性 = 基础值 × (1 + 升级倍率 × (等级 - 1))
```

### 升级倍率
- HP：5% = 0.05
- Attack：3% = 0.03
- Defense：3% = 0.03
- Speed：0（固定1.0）
- CritRate：0（固定5%）
- CritDamage：0（固定150%）

### 示例计算（B001铁壁30级）
```
HP = 180 × (1 + 0.05 × 29) = 180 × 2.45 = 441
Attack = 18 × (1 + 0.03 × 29) = 18 × 1.87 = 34
Defense = 25 × (1 + 0.03 × 29) = 25 × 1.87 = 47
Speed = 1.0（固定）
CritRate = 5%（固定）
CritDamage = 150%（固定）
```
