## 任务流程图

```
支线任务（引导）- 独立进行：
guide_001 → guide_002 → guide_003 → guide_004 → guide_005

主线第一章 - 独立进行：
main_ch1_001 → main_ch1_002 → main_ch1_003 → main_ch1_004 → main_ch1_005 → main_ch1_006
                                                                        ↓
主线第二章：
main_ch2_001 → main_ch2_002 → main_ch2_003 → main_ch2_004 → main_ch2_005

日常任务（独立，每日重置）：
daily_001, daily_002, daily_003, daily_004, daily_005

注意：主线任务和支线任务相互独立，可同时进行！
```
