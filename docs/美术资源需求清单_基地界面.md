## 七、动画需求（如支持）

### 24. 循环动画
1. **能量流动**：蓝色光点在管道中流动（2秒循环）
2. **星图旋转**：星空缓慢旋转（30秒循环）
3. **熔炉火焰**：火焰跳动效果（1秒循环）
4. **扫描线**：从上到下扫描（3秒循环）
5. **呼吸灯**：亮度变化（2秒循环）

### 25. 交互动画
1. **按钮悬停**：放大+发光增强（0.3秒）
2. **面板展开**：从中心展开（0.5秒）
3. **收益飘字**：向上飘动+淡出（1秒）
4. **解锁动画**：粒子爆发+光芒（1秒）
