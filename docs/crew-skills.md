# 船员技能设计文档

## 概述

每个角色拥有4个独特技能：

- **普攻**：基础攻击，无冷却
- **主动技能**：特殊技能，有冷却时间
- **终极技能**：强力技能，无冷却，需要能量
- **天赋**：被动技能，自动生效

技能等级可以升级，升级后技能效果会增强。

- **普攻**：基础可提升至5级
- **主动技能**：基础可提升至4级，第5级依靠星级
- **终极技能**：基础可提升至4级，第5级依靠星级

## 能量系统

- **普攻**：+5点能量
- **主动技能**：+20点能量
- **终极技能**：消耗100点能量

## 技能效果汇总

### 伤害类型

- **物理**：普通物理攻击
- **能量**：能量武器攻击
- **爆炸**：爆炸伤害，通常有溅射效果
- **腐蚀**：持续伤害，通常附带中毒效果
- **电击**：电击伤害，通常附带控制效果
- **真实**：无视防御的伤害

### 控制效果

- **眩晕**：无法行动
- **沉默**：无法使用技能
- **冰冻**：无法行动，受到火属性伤害翻倍
- **睡眠**：无法行动，受到伤害后醒来
- **混乱**：攻击随机目标（可能攻击队友）
- **失明**：命中率大幅降低
- **束缚**：无法使用普攻
- **封印**：无法使用终极技能

### 增益效果

- **护盾**：吸收伤害的屏障
- **攻击提升**：攻击力增加
- **防御提升**：防御力增加
- **速度提升**：攻速增加
- **暴击提升**：暴击率增加
- **治疗提升**：受到的治疗效果增加
- **无敌**：免疫所有伤害
- **隐身**：无法被选中

### 减益效果

- **攻击降低**：攻击力减少
- **防御降低**：防御力减少
- **速度降低**：攻速减少
- **中毒**：每回合损失生命
- **灼烧**：每回合损失生命
- **流血**：每回合损失生命
- **标记**：受到的伤害增加
